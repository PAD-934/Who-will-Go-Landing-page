## 🆘 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Orders not in Google Forms | Check entry IDs, verify form is accepting responses |
| Email not received | Check spam folder, email is secondary system |
| JavaScript errors (F12) | Usually wrong entry IDs - double check |
| Nothing happens on submit | Make sure script.js was saved after editing |

**More help:** Check browser console (F12) when testing
